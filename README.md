# Новое русское вино

Сайт магазина авторского вина "Новое русское вино".

## Установка и запуск

### 1. Клонируйте репозиторий
``` bash
git clone https://github.com/Suba79/wine-shop-project.git
cd wine-shop-project
``` 
### 2. Создайте виртуальное окружение

``` bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS / Linux
python3 -m venv venv
source venv/bin/activate
```

### 3. Установите зависимости

``` bash
pip install -r requirements.txt
```
### 4. Подготовьте файл с данными

В корне проекта уже есть файл wine_catalog.xlsx с актуальным ассортиментом.
При желании вы можете заменить его своим файлом (сохранив структуру колонок).

1. Скопируйте файл-образец:
```bash
cp wine_catalog.xlsx my_wines.xlsx   # для macOS/Linux
copy wine_catalog.xlsx my_wines.xlsx  # для Windows
```
2. Откройте `my_wines.xlsx` в Excel или любом редакторе таблиц

3. Запустите программу с указанием пути к вашему файлу:

``` bash
python main.py --data-path my_wines.xlsx
```

### 5. Запустите генерацию сайта и сервер

``` bash
python main.py
```

### 6. Проверьте результат

* Откройте браузер и перейдите по адресу: http://127.0.0.1:8000/index.html

### Структура Excel-файла

Файл `wine_catalog.xlsx` должен содержать следующие колонки:

| `Колонка`   | Описание                              | Пример                               |
|----------|----------------------------------------|--------------------------------------|
| `Категория` | Группа товара                         | Белые вина, Красные вина, Напитки    |
| `Название`  | Название продукта                      | Белая леди, Чача, Хванчкара          |
| `Сорт`      | Сорт винограда (может быть пустым)     | Дамский пальчик, Александраули       |
| `Цена`      | Цена в рублях (только число)           | 399, 299, 550                        |
| `Картинка`  | Имя файла изображения                  | belaya_ledi.png, chacha.png          |
| `Акция`     | Отметка об акции (оставьте пустым, если нет) | Выгодное предложение          |

### Пример заполнения:

| Категория     | Название     | Сорт             | Цена | Картинка         | Акция                 |
|--------------|--------------|------------------|-----:|------------------|-----------------------|
| Белые вина    | Белая леди   | Дамский пальчик  |  399 | belaya_ledi.png  | Выгодное предложение  |
| Напитки       | Чача         |                  |  299 | chacha.png       | Выгодное предложение  |
| Красные вина  | Хванчкара    | Александраули    |  550 | hvanchkara.png   |                       |

### Примечания по заполнению
* Сорт можно оставить пустым для напитков (чача, коньяк)
* Акция заполняется только для товаров со скидкой, значением `Выгодное предложение`
* Все изображения должны быть в папке `images/` в формате PNG или JPG
