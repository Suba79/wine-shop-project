# Новое русское вино

Сайт магазина авторского вина "Новое русское вино".

## Установка и запуск

### 1. Клонируйте репозиторий
``` bash
git clone https://github.com/Suba79/wine-shop-project.git
cd wine-shop-project
``` 
### 2. Создайте виртуальное окружение

``` bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS / Linux
python3 -m venv venv
source venv/bin/activate
```

### 3. Установите зависимости

``` bash
pip install -r requirements.txt
```
### 4. Подготовьте файл с данными

В корне проекта уже есть файл wine_catalog.xlsx с актуальным ассортиментом.
При желании вы можете заменить его своим файлом (сохранив структуру колонок).

### 5. Запустите генерацию сайта и сервер

``` bash
python main.py
```

### 6. Проверьте результат

* Откройте браузер и перейдите по адресу: http://127.0.0.1:8000/index.html
* Вы должны увидеть страницу с категориями вин и карточками товаров
* В терминале будет выведена информация о загруженных винах и товарах с акцией

### Структура Excel-файла

Файл wine_catalog.xlsx должен содержать следующие колонки:

* `Категория` — например, "Белые вина", "Красные вина", "Напитки"
* `Название` — название продукта
* `Сорт` — сорт винограда (может быть пустым для напитков)
* `Цена` — цена в рублях
* `Картинка` — имя файла изображения (должно лежать в папке images/)
* `Акция` — значение "Выгодное предложение" для товаров со скидкой
